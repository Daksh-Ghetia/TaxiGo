# Install node modules for backend in the backend file

## use this command in both the files frontend and backend as well
npm intall -- force

## Angular 14 is required for project if not working install it using command mentioned below
npm install -g @angular/cli@14

## To start the backed
 - go to backend folder
 - run `npm run dev`

## To start the frontend
 - go to frontend folder
 - run `ng serve`
